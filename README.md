# Project: AzCloud

## Requirements
- **docker** and **docker-compose**
- **make** executable

## Initialization
- `git clone the repository` && `cd src/`
- `copy .env.example to .env`
- `make init`

## Synchronization of flavors
- `make sync`

# Notes
- **SQL dump** location - ./src/azcloud-data.sql (You need import this after initialization)
